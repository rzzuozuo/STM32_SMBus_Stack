Update History

  

V2.0.3 / 2-August-2018

 

Main Changes

•  Bug fixes, updated with new test vectors. MISRA cleanup. PEC/ARP has separate defines now.

 

Content

•  STM32 SMBus 3.0 and PMBus 1.3.1 stack

 

Development Toolchains and Compilers

•  IAR Embedded Workbench 8.22.2

•  Keil MDK 5.24.

 

Supported Devices and EVAL boards

•  Tested with STM32L0538-Discovery (MB1143B, embedding STM32L053C8 device).

•  Tested with STM32L476G-Discovery (MB1184C, embedding STM32L476VG device).

•  Tested with STM32F3-Discovery (MB1035B, embedding STM32F303VC device).

•  Tested with STM32WB-Nucleo (MB1291B, embedding STM32WB55VG device).

•  Tested with STM32H7-Nucleo (MB1137, embedding STM32H743ZI device).

•  Tested with STM32G4-Nucleo (MB1367C, embedding STM32G431RB device).

•  The stack is designed to work on any STM32 device where SMBus HAL is available, however adjustments may be required.

 

Known Limitations

•  Only a workaround transmitting FFs prevents from a bus conflict on UDID read during ARP when there are more devices on the bus.
